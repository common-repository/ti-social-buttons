A simple and minimal plugin to add facebook and twitter buttons at the end of pages and posts.
== Frequently Asked Questions ==

= What's the difference between this plugin and its alternatives =

The main difference is that this plugin does not inserts any intermediate url. It share the original url from your wordpress website.
